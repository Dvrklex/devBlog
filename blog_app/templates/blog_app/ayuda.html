Guía de Markdown

Markdown es una sintaxis de marcado simple que te permite formatear el texto de una manera más fácil y rápida. Aquí tienes algunos ejemplos de cómo puedes usar Markdown:

Encabezados

# Encabezado de nivel 1
## Encabezado de nivel 2
### Encabezado de nivel 3

Negrita e Itálica

**Texto en negrita**
*Texto en itálica*

Listas

- Elemento de lista 1
- Elemento de lista 2
- Elemento de lista 3

Listas Numeradas

1. Elemento 1
2. Elemento 2
3. Elemento 3

Enlaces

[Texto del enlace](URL_del_enlace)

Imágenes

![Texto alternativo](URL_de_la_imagen)

Código en línea y Bloques de código

`Código en línea`

Citas

> Esto es una cita.

Línea horizontal

---

Recuerda que puedes utilizar Markdown en tus posts para darles un formato más elegante y legible.
